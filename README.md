# ⚽ Mundialito Bethshalom

¡Bienvenidos al repositorio oficial del **Mundialito Bethshalom**! Este proyecto fue diseñado específicamente para organizar y dar vida a un torneo de fútbol de nuestra comunidad educativa promovida por la promoción INVICTUS y su personero Juan Pablo Mendoza Villamizar. 

No es una aplicación comercial; es una herramienta creada desde cero para que los visitantes puedan seguir cada detalle de la competición en tiempo real.

## 🌟 ¿Qué se puede hacer con esta App?
* **Seguir el ritmo del torneo:** Consultar los próximos partidos y los resultados de los encuentros ya jugados.
* **Ver quién manda en la tabla:** Las posiciones se actualizan automáticamente para saber quién lidera el torneo.
* **Goleadores y disciplina:** Llevar el conteo de quiénes son los "pichichis" y quiénes deben cuidarse de las tarjetas amarillas y rojas.
* **Gestión transparente:** Como administrador, puedo cargar los goles y eventos de cada partido para que la información sea inmediata y veraz.

## ⚠️ Observaciones y Restricciones del Torneo
Es importante recordar que este código está adaptado a la lógica propia de nuestro evento institucional. Por lo tanto, tiene algunas condiciones especiales:
1.  **Calendario Institucional:** La programación de los partidos respeta los horarios escolares, calendario institucional y las restricciones de días festivos en Colombia.
2.  **Uso Académico y Deportivo:** Aunque el código está disponible aquí, fue diseñado con un propósito pedagógico y deportivo para la institución, por lo que su configuración actual es muy específica para nuestra estructura de equipos y categorías.
4.  **No Comercial:** Esta es una creación propia con fines recreativos y educativos, por lo que no debe ser usado con intención de comercializar u obtener beneficios .

## 🛠️ Un poco sobre la "magia" detrás
Para que esto funcione sin problemas, utilicé:
* **HTML, CSS y JavaScript** para que se vea bien y sea fácil de usar desde el celular.
* **Firebase** como el motor que guarda todos los datos (goles, tarjetas y posiciones) de forma segura y rápida.
